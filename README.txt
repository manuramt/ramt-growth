Instrucciones rápidas para reemplazar imágenes:
1. Reemplaza los archivos en /img/ con tus imágenes reales (mismo nombre).
   - hero.jpg: widescreen ~1600x800 (JPEG). Usar imagen en BW o aplicar filtro grayscale.
   - avatar*.png: cuadrados ~800x800 (PNG/JPG). Preferible avatares IA estilo corporate.
   - funnel.png: ilustración o diagrama ~1000x600.
2. Opcional: modificar el color de acento editando --accent en style.css.
3. Para publicar en GitHub Pages:
   - Crea un repo público.
   - Subir todos los archivos en la rama main.
   - En Settings > Pages, servir desde root (main).
   - Conecta el dominio ramt.es creando un CNAME o A records (si quieres, te doy los pasos exactos).

Sugerencias de imágenes AI por bloque:
- Hero: Avatar IA profesional + background workspace minimal
  Prompt ejemplo: "corporate AI avatar, neutral expression, soft studio lighting, minimal background, photoreal stylized"
- Value: dashboard UI mockup, simple metric cards
- Services (Agents): chat UI + avatar in conversation bubbles
- Services (Funnel): clean funnel diagram with 3 stages, labels, subtle gradients
- Process: line icons set, coherent stroke weight (2px)
- Proof/Metrics: simple numeric tiles and microcopy
